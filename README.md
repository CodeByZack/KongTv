## 功能介绍 ：

- 支持观看记录。
- 支持收藏。
- 支持多线路切换（虽然视频站挂的没剩下几个了）
- 支持投屏播放（不保证都能用）
- 支持多种播放模式切换
- 支持在线更新

## 使用到的开源项目：

- 视频播放-NiceVieoPlayer
- 投屏播放-Cling
- 网络请求-Okhttp+rxjava2+retrofit2
- 网页解析-Jsoup
- 网页播放-AgentwebX5
